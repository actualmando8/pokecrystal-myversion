## Required Files for Each Pokémon

Each folder should contain:
- `front.png` - Front sprite image
- `back.png` - Back sprite image
- `front.animated.2bpp.lz` - Compiled front sprite (auto-generated on build)
- `back.2bpp.lz` - Compiled back sprite (auto-generated on build)
- `normal.gbcpal` - Normal palette (auto-generated from front.png)
- `shiny.pal` - Shiny palette definition
- `anim.asm` - Animation script
- `anim_idle.asm` - Idle animation script
- `bitmask.asm` - Sprite bitmask data
- `frames.asm` - Frame data
- `front.dimensions` - Sprite dimensions

## After Downloading

1. Make sure all files are in the correct directories
2. Run `make` to compile the ROM
3. The build system will automatically generate the .2bpp.lz and .gbcpal files from the .png files

## Troubleshooting

If you get compilation errors:
- Ensure all files are downloaded
- Check that folder names match exactly (lowercase, underscores)
- Verify the .png files are valid images
- Make sure the .asm files have proper syntax
